# 🔥 Pokédex Web

Aplicação web desenvolvida em **React** que consome dados da **PokeAPI**, permitindo visualizar, buscar e filtrar Pokémons de forma dinâmica.

---

## 🚀 Tecnologias utilizadas

- React
- JavaScript
- CSS
- PokeAPI
